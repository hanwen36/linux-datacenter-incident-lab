# Disk Full Incident Lab

## Objective
Simulate a disk full condition and troubleshoot it.

## Environment
Ubuntu Server
VirtualBox

## Commands Used
df
du
sort
rm
fallocate

## Steps
1. Check disk usage
2. Create large files
3. Identify large files
4. Remove unnecessary files
5. Verify disk recovery
